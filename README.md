# FilIMSS
La fila de la Salud
“FILIMSS” es un proyecto que se tiene como objetivo:
Mejorar las funciones del personal de salud involucrado en la lista de espera mediante el avance tecnológico, empleando un sitio web para facilitar la organización de las citas médicas otorgadas de los derechohabientes, el aviso de aproximación de lugar para citas  médicas  así como facilitar y brindar información sobre temas a fines y subsecuentes de la atención médica.
Objetivos específicos: 
I.- Generar un mejor control y organización de los derechohabientes. 
II.- Fomentar una postura de comunicación entre derechohabientes y personal de salud involucrado.
